# Semester 4 Workspace

This repository organizes all Semester 4 subjects and related study material in one place.

## Structure

Each subject folder generally follows this format:

- Assignments
- Modules
- Notes
- Suggestion
- Syllabus

Some subjects may only include the folders relevant to that course.

## Subject Index

- 00_Admin/

## Quick Navigation

- Admin resources: `00_Admin/`
- Subject-wise materials: open the corresponding subject folder
- Assignment-first workflow: start from each subject's `Assignments/` folder
- Exam preparation workflow: review `Modules/`, then `Notes/`, then `Suggestion/`, and finally `Syllabus/`

## Suggested Usage

1. Keep all new class files inside the matching subject folder.
2. Place submission-ready work in `Assignments/`.
3. Keep reference reading material in `Notes/`.
4. Store official course outline updates in `Syllabus/`.

## Notes

- Folder names are intentionally course-specific to match your academic structure.
- You can later add per-subject README files if you want detailed tracking (deadlines, status, links, and revision plans).
